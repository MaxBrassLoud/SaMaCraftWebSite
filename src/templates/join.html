Discord beitreten & Whitelist

Um Spam zu verhindern, ist eine Whitelist aktiv. Du musst dich zuerst auf unserem Discord anmelden lassen.

Tritt unserem Discord bei und melde dich für die Whitelist an: discord.gg/E3ZHhWyfMG

3
Minecraft starten
  1. Starte den Minecraft Launcher.
  2. Stelle sicher, dass die Java Edition ausgewählt ist.
  3. Wähle Version 1.21.10 oder 1.21.11 – installiere sie falls nötig über „Installationen".
  4. Klicke auf Spielen.
4
Server hinzufügen & beitreten
  1. Klicke im Hauptmenü auf Mehrspieler.
  2. Klicke auf Server hinzufügen und fülle ein:
Servername
SaMaCraft
Serveradresse
mc.samacraft.de
  1. Klicke Fertig, wähle den Server und klicke Beitreten.
5
Tipps für neue Spieler
📖 Lies zuerst das Regelwerk – damit vermeidest du Probleme von Anfang an.
🏠 Nutze /spawn, falls du dich verlaufen hast.
🤝 Respektiere andere Spieler – Fairplay macht mehr Spaß!
🗺️ Erkunde den Server und entdecke spannende Gebäude und Events.
